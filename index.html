Was ist ein Stabilitätsbonus?

Bei LoRaWAN zählt ein stabiles Netz oft mehr als eine sehr gute Signalstärke.

Messpunkte erhalten einen Bonus, je nachdem wie viele der 5 vorherigen Messpunkte desselben Boards im 175m-Radius liegen und erfolgreich empfangen wurden.

Bonus-Skala:

  • ✅ 5/5 Pings: +15 dB
  • ✅ 4/5 Pings: +10 dB
  • ✅ 3/5 Pings: +5 dB
  • ✅ 2/5 Pings: +2 dB
  • ✅ 1/5 Pings: +1 dB
  • ❌ 0/5 Pings: +0 dB
×

Treiber für Windows

Unter Windows wird ein passender USB-Treiber benötigt.

  1. Lade den CP210x VCP Windows Treiber bei Silicon Labs herunter (unter "Downloads").
  2. Entpacke die ZIP-Datei und starte die passende .exe-Datei
  3. Ob x64 oder x86 hängt von deinem System ab. Probiere einfach beide - der falsche Treiber kann nicht installiert werden.
  4. Starte den Browser danach ggf. neu.
